Final Year Honours Project

Blender, Unity & Adobe Suite - May 2025

An interactive hybrid/2.5D animated user interface running in the Unity engine, blending custom 3D and 2D components in a consistent comic-book inspired style. Guided by functional and non-functional requirements, the project progressed through concept development, implementation and iteration based on user-testing feedback to create a fully functional interface with animated transitions between screens. Partially evaluated on successful implementation of accessibility considerations, e.g. for those with visual impairments. Applied HCI and user-centred concepts to design for the user and guide best practices for GDPR-complaint collection of participant data.
