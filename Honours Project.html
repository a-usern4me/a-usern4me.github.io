Final Year Honours Project

Blender, Unity & Adobe Suite - May 2025

A dynamic and interactive user interface running in the Unity engine, using cel shading to seamlessly combine custom 3D and 2D components in a consistent comic-book style. Features animated transitions to each screen using the character model which was created, rigged and animated in Blender.
